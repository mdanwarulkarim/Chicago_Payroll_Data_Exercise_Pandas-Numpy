# City of Chicago Payroll Data EDA

## Project Overview
This project explores the **City of Chicago Payroll Data** using Python, Pandas, and NumPy. The goal is to clean, analyze, and summarize payroll information in a clear and reproducible Jupyter Notebook workflow.

## Main Points
- **Data cleaning**: Converts salary and hourly-rate values from strings like `"$107,790.00"` into numeric values for analysis.
- **Statistical analysis**: Calculates average, maximum, and minimum salaries and hourly rates.
- **Department insights**: Identifies which departments have the highest employee counts, especially **Police and Fire**.
- **Employee filtering**: Searches for specific employees, unique job titles, and full-time vs. part-time staff.

## Additional Features
- Supports exploratory data analysis with Pandas and NumPy.
- Helps compare workforce distribution across departments.
- Makes it easier to inspect salary patterns and role types.
- Can be extended with charts, dashboards, or deeper demographic analysis.

## Workforce Demographics
The dataset shows that the **Police Department is the largest employer** in the city, with **over 12,000 employees**, followed by the Fire Department. Most of the workforce consists of **full-time, salaried employees**.

## How to Use the Jupyter Notebook
### 1. Environment Setup
Make sure Python is installed. It is recommended to use Anaconda or install Jupyter with:
```bash
pip install notebook
```

### 2. Launch Jupyter Notebook
Open your terminal or command prompt, go to the project folder, and run:
```bash
jupyter notebook
```

### 3. Open the Notebook
When the notebook interface opens in your browser, click:
`Karim_Chicago_Payroll_Data_Exercise.ipynb`

### 4. Run the Cells
Use **Shift + Enter** to run each cell. You can modify the dummy data or logic to test different scenarios.
